1//! Enables convertion between Esri geometry types and geometries from [`geo_types`]   
2//!
3//! # Coord
4//! [`geo_types::Coord`] can be converted into any of [`Coord`](crate::geometry::Coord)  ( [`CoordXy`](crate::geometry::CoordXy), [`CoordXyz`](crate::geometry::CoordXyz), [`CoordXym`](crate::geometry::CoordXym), [`CoordXyzm`](crate::geometry::CoordXyzm) )  
5//!  - Missing Z coordinates are filled with `0.0_f64` and missing M coordinates are filled with `None`  
6//!
7//! [`Coord`](crate::geometry::Coord) can be converted into [`geo_types::Coord`]  
8//!  - M and Z coordinates are ignored  
9//!
10//! These changes apply to the coordinates when converting all other geometries
11//!
12//! # Points
13//! Point types are easily convertible  
14//! [`geo_types::Point`] and [`Point`](crate::geometry::Point) can be converted to each other  
15//!
16//! [`geo_types::MultiPoint`] and [`MultiPoint`](crate::geometry::MultiPoint) can be converted to each other  
17//!
18//! # Lines
19//! [`geo_types::Line`], [`geo_types::LineString`] and [`geo_types::MultiLineString`] can be converted to [`LineString`](crate::geometry::Polyline)  
20//!
21//! [`LineString`](crate::geometry::Polyline) can only be converted to [`geo_types::MultiLineString`]  
22//!
23//! # Polygons
24//! [`geo_types::Polygon`], [`geo_types::MultiPolygon`], [`geo_types::Rect`] and [`geo_types::Triangle`] can be converted to [`Polygon`](crate::geometry::Polygon)  
25//!
26//! [`Polygon`](crate::geometry::Polygon) can only be converted to [`geo_types::MultiPolygon`]  
27//!
28//! # Geometry
29//! [`geo_types::Geometry`] can mostly be converted to [`Geometry`](crate::geometry::Geometry), but [`geo_types::GeometryCollection`] cannot be converted
30//!
31//! [`Geometry`](crate::geometry::Geometry) can only be converted to [`geo_types::Geometry`]
32//!
33//! # Spatial Reference
34//! The [`SpatialReference`](crate::geometry::SpatialReference) type is ignored when converting and will need to be set separately  
35//!
